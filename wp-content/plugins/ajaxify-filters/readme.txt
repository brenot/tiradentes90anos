Ajaxify your filters without wasting time in page load

== Description ==

"Ajaxify Filters" provides you widgets that works as a replacement to default WooCommerce-widgets and lets you catch your products on-the-fly without page load using AJAX technique.

= Features =

* Provide widget to **filter product category wise**.
* Provide widget to **filter product tag wise**.
* Provide widget to **filter product price wise**.
* Provide widget to **filter product attribute wise**.
* Provide widget to **type product name and search them**.
* Provide widget to **show active filters to remove them easily**.
* Provide widget to **save active filters to apply same filters again very easily**.
* **All filters works in accordance with each-other**, i.e., when one filter is applied it eliminates all unnecessary filters options, giving user the option to filter products in a chain-wise manner.
* **Provides 4 different layout** of showing widget in frontend for product attributes which is set by admin :
	**(a)List**
	**(b)Checkbox**
	**(c)Color**
	**(d)Label**
* **Provides SEO friendly URLs**.
* **Applys AJAX to woocommerce-pagination as well woocommerce-sorting**.
